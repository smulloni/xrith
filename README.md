# xrith — cross-rhythm explorer

This web-based polyrhythm tool is essentially a fancy metronome that plays 
and visualizes multiple integer subdivisions of one shared loop — hear and 
see 4-against-7, 3:4:5, and arbitrary stacks, with each layer given a distinct 
voice with a concentric ring visualization.

**▶ Live: https://smulloni.github.io/xrith/**

Zero build, no dependencies — plain ES modules. Open `index.html` locally, or
run the tests with `npm test` (`node --test`).
